# How to see the App ? 
1. Clone the repository 
```sh
git clone https://github.com/yogita10/Photo_Clicking_App.git
```
3. Go to the clone folder
```sh
cd Photo_Clicking_App
```
5. Run the following commands: 
```sh
 npm install 
```
```sh
 npm run android 
```
